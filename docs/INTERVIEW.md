# Mock Interview Simulator

The mock interview simulator lives within the Coach Center page and provides an AI‑powered, resume-and-job-driven interview experience. Users choose a resume, a saved job, and a role level (entry/mid/senior), then answer a series of questions generated by an LLM. Each response is evaluated and a summary is stored.

## Backend Endpoints

- `POST /api/interview/start`
  - Body: `{ resume_id: string, job_apply_url: string, role_level: string }`
  - Returns first question and session ID.
- `POST /api/interview/respond`
  - Query params: `session_id`, `answer`
  - Returns evaluation, next question, and completion flag.
- `GET /api/interview/session/{session_id}`
  - Retrieves full session history and summary.

## Database Schema

- `InterviewSession` nodes with properties: `session_id`, `resume_id`, `resume_name`, `job_apply_url`, `job_title`, `job_company`, `job_requirements`, `job_responsibilities`, `role_level`, `started_at`, `completed_at`.
- `InterviewStep` nodes linked via `HAS_STEP` and holding `question`, `answer`, `evaluation_score`, `evaluation_feedback`, `timestamp`.

## Frontend Notes

- `nextjs/hooks/useInterview.ts` manages state, persistence and API calls.
- `nextjs/lib/interviewApi.ts` exports API helpers.
- Coach Center page (`nextjs/app/coach-center/page.tsx`) embeds the chat UI.
- State is saved to `localStorage` under `careerlift:interview`.
